＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

この作品は昔はエヴァレインだった女と一部のグレアムの要素を持つ男が夫婦としてかウィームで暮らす話です。ヒロインの名前まだ決めてないです。

頑張って原作との矛盾や乖離などを回避します、原作の展開に合わせ常に更新して必要の時は書き直したりするつもりです。

原作から作った仮説になりますが。

グレアムが含め思い出せない事いくつある（継続１４５）のと、
再会（解雇５８２、６８２）してから燭台の塔でディノとの会話（解雇２８４）を覚えている記述がない。

解雇９６１でグレアムが言った、妻の喪失にずたずたにされた心と心の別の部屋のディノに向けると執着や愛が上げる悲鳴（ディノを一人にしない願い）、

「それは、エヴァレインを愛し、その喪失にずたずたにされた心とはまた別の、グレアムの心の別の部屋に残された執着や愛が上げる悲鳴だったのだろう。」
「どれだけを失い、この世界にはもう胸が張り裂けそうなくらいに愛おしい妻がいなくても。
ああ そうだ。彼女がいなくても、自分は生きてはいける。」

やや強引な解釈になりますが、私はディノのそばに戻ったグレアムが取り戻しの魔術にエヴァレインへの執着を生きていられる程度に一部削り落とされたかもしれないと考えている。
その執着と、一部欠けている記憶や公爵から侯爵への階位落ちによって無くした魔術なと（グレアムの階位は後天的上がった物だからかもしれないのですが）がグレアムではない誰かを生み出すかもしれないと思った次第です。

ここでエルトとフェルフィーズを参考して、エヴナがフェルフィーズはエーヴァルトではないと判断した事（解雇９６０）と魔物が生まれ変わりを同一人物として認識しない事を考え、

「最後にその資質を大きく欠いた魔物は、次の派生の時に先代の要素を引継ぎ難い。
狂乱し鎮められたグレアムは、もう自分のよく知る彼ではないと、どこかで諦観していた。」（解雇６８８）

エヴァレインが狂乱した上にウイリアムと交戦したので資質を大きく欠いた事て新しい戦火の魔物は容姿も性格はエヴァレインと違う可能性が高いかもしれない。

なのでエヴァレインの魂の行き先だと、魂が砕けられたエーヴァルトとエヴナの契約の子供だったヴェルリア王も転生しているので（解雇６６１）。

１、戦火の魔物だけと前世と似てない。
２、戦火の魔物じゃない魔物。
３、魔物じゃない。

といった感じですね。

小ネタですがヒロインの髪は浅緋色て、緋色だったエヴァレインと少し印象が違う。
口調や性格、趣味嗜好にも違いがあります。

実はオリヒロインを書いてる途中で何だか性格も見た目もエヴァレインに似ているなって、語り手しゃべり方もグレアムに似ているのでテキストの大半を書き直して今の物にしました。
その時はヒロインの髪色も合わせて浅緋色に修正しました。
