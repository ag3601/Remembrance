今の設定

メインヒロインは芥子の魔物のパッチェです（オリキャラ）、ユーザ（パソコン前のあなた）は彼女の歌乞いで伴侶。

ウィーム中央に住んでます。

植物と終焉の系譜、薬も取り扱う。アルビクロムとガーウィン、カルウィにある程度の影響力を持つ古い魔物の一柱。

ちなみに植物の系譜のヒロインにしたのはエマジュリア好きなのて。

名前の元ネタはラテン語Requiescat in pace（安らかに眠れ）の教会ラテン語読み

芥子はギリシャ、ローマを含む古代地中海の文化とそれを継承する西欧文化では安らかさ、眠りを象徴するようです。

ユーザー以外の相方を入れる場合、相方の名前は芥子のシーのティアにする予定。元ネタはラテン語Retinentia（追憶）。

ティア向きなシェルがなかなか見つからない。
